# 📊 Dashboard Analisis Nilai Mahasiswa

Aplikasi berbasis **Streamlit** untuk menganalisis performa mahasiswa (M1-M4) menggunakan **Hybrid Forecasting** dan **Machine Learning (Random Forest)**.

---

## 🚀 Fitur
- Upload data CSV/XLSX
- Analisis nilai mahasiswa
- Evaluasi model (MAE, RMSE, R²)
- Visualisasi (histogram, scatter, korelasi)
- Insight ekonomi informasi
- Agent pembelajaran interaktif

---

## 🛠️ Teknologi
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

---

## ▶️ Cara Menjalankan
```bash
pip install -r requirements.txt
streamlit run app.py
